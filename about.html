What you can do

Flashcards from CSV

Upload a two‑column CSV (term,definition) and start studying immediately.

Flexible content

Each entry can be text or a media link (image / YouTube). Cards will display the right format automatically.

Professional UI

Clean layout, keyboard support, and appearance that follows system settings.

CSV format

term,definition
Heap,A specialized tree-based data structure
Quicksort,Divide-and-conquer sorting algorithm

Only two columns are required: term and definition. If a cell contains a media link instead of text, the card will render it.

How to use Accelerate

  1. Open Accelerate.
  2. Upload your CSV or build one with the CSV Deck Builder.
  3. Study using the keyboard: Space to flip, ←/→ to navigate.
